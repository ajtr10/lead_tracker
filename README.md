
# Lead Tracker — Chrome Extension

Simple Chrome extension to quickly save links. Built with **HTML, CSS, JS** (Scrimba).

## Features
- **Save URL** — add a link manually  
- **Save Tab** — save the current tab's URL with one click  
- **Delete All** — clear all saved links (localStorage)

## Quick start
1. `git clone https://github.com/yourusername/lead_tracker.git`  
2. Open Chrome → `chrome://extensions/` → enable **Developer mode**  
3. Click **Load unpacked** → select the project folder  
4. Use the extension: Save URL / Save Tab / Delete All
